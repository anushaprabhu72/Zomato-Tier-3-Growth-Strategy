# Zomato-Tier-3-Growth-Strategy
Strategic consulting project analyzing growth opportunities for Zomato in Tier-3 Indian cities, and under what conditions it can do so without compromising consolidated profitability.

Overview
This project synthesizes primary financial data from Eternal's FY25 Annual Report, competitive intelligence, and market sizing frameworks to develop a phased, cluster-based go-to-market recommendation for Zomato's non-metro expansion.

Key Areas Covered
•	Market Sizing  — Serviceable Tier 3 demand pool sizing, TAM/SAM analysis
•	Financial Baseline  — FY23–FY25 food delivery metrics, war chest analysis, breakeven modeling
•	Competitive Landscape  — Swiggy, ONDC, Amazon/Flipkart, and regional player positioning
•	Unit Economics  — Cost structure comparison: Tier 1 vs. Tier 3 (delivery cost, CAC, AOV, dark store capex)
•	SWOT Analysis  — Strategic strengths, weaknesses, and growth levers
•	3-Year Roadmap  — Phase-gated rollout from 50 pilot cities (FY26) to 500+ cities (FY28)
•	Risk Mitigation  — Order density, competitive response, AOV compression, and macro risks
